Lagrange Interpolation

finds a unique polynomial of degree \(n - 1\) that passes through a given set of \(n\) data points. This polynomial is constructed by summing weighted Lagrangian basis polynomials.
Formula:
\[P(x)=\sum_{i=0}^ny_iL_i(x)\] where: \[L_i(x)=\prod_{j=0,j\neq i}^n\frac{x-x_j}{x_i-x_j}\] Langrange Interpolation can be limited because high-degree interpolations can have very significant oscillations between data points, which means that it can be highly unreliable for extrapolation.

Newton's Divided Difference Interpolation

finds the same polynomial as Lagrange interpolation, but through a difference method. This method uses divided differences to calculate coefficients \(a_i\) for the polynomial \[P(x)=f[x_0]+\sum_{i=0}^nf[x_0,\dots,x_i]\prod_{j=0}^{i-1}(x-x_j)\] where \[a_0=f[x_0], a_1=f[x_0,x_1],\dots, a_n=f[x_0,x_1,\dots,x_n]\] where \(f[x_i]\) is a divided difference. Some examples of divided differences are: \[f[x_i]=y_i\] \[f[x_i, x_{i+1}=\frac{f[x_{i+1}] - f[x_i]}{x_{i + 1} - x_i}\] \[f[x_i, x_{i + 1}, \dots, x_{i + k}] = \frac{f[x_{i + 1},\dots,x_{i + k}] - f[x_i,\dots,x_{i + k - 1}]}{x_{i + k} - x_i}\] Because this generates the same polynomial as Lagrange interpolation,

Cubic Spline Interpolation

constructs a smooth piecewise cubic polynomail function \(S(x)\). \(S(x)\) passes through a set of points and ensures continuity of the function, the first derivative of the function, and the second derivative of the function at each interior "knot".
\(S(x)\) consists of \(n-1\) cubic polynomials given \(n\) points. Each separate cubic polynomial is of the form: \[S_i(x)=a_i+b_i(x-x_i)+c(x-x_i)^2+d_i(x-x_i)^3\] To create these cubic polynomials, a tridiagonal linear system is solved to find the coefficients.
Using a cubic spline can be better because it reduces the risk of oscillation and makes the interpolation smoother overall.

Hermite Interpolation

is similar to cubic spline interpolation, but instead of having conitnuity up to the second derivative, it only has continuity up to the first derivative.
The goal is to find a polynomial \(H(x)\) of the lowest degree that is continuous and the first derivative is continuous. The calculation involves using either the Lagrange Basis polynomials \(L_i(x)\) or the divided differences method from earlier. Using the Lagrange basis polynomials, the polynomial can be constructed like: \[H(x)=\sum_{i=0}^nf(x_i)h_i(x)+\sum_{i=0}^nf'(x_i)\hat{h}_i(x)\] Hermite Interpolation is less smooth than Cubic Spline interpolation, which has the potential to introduce more wiggles in the function.
Usually, it makes sense to choose Hermite Interpolation when preserving monotonicity, preventing overshooting, or when local control is more important than the overall smoothness of the function.

Picking Interpolation Points


Another important variable to consider when interpolating the data is picking the right points to use for interpolation. For this project, I was experimenting with the following methods of picking points:

Random

  • This method will usually suck and will usually be the worst of the three and will serve as a good baseline

Equally-Spaced

  • Models based on observational data coming in at discrete time scales might require this method to be used.
  • It is simple to calculate and a straightforward sampling method to understand.
  • It is prone to oscillations and instability

Chebyshev

  • The gold standard for polynomial interpolation, so I expected this to perform the best.
  • Mathematically, on the interval \([-1,1]\) the points are defined as:
    • \[x_i=\cos\big{(}\frac{2i+1}{2n+2}\pi\big{)}\]
  • This method minimizes the maximum error. This is because the spacing of the points creates greater clustering at the edges of the interval and can prevent oscillation.
  • However, since the datapoints being used are discrete, this method can't be used directly so this will make the method slighlty less reliable.
