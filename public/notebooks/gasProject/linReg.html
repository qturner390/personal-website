Interpretting the Inital Linear Regression Results:

The initial model shows pretty significant multicollinearity, which is clear given CPI's high variance inflation factor of 9.36. This means that the collinearity inflated the standard errors and might mask the actual input of the other variables in the OLS regression.
Becuase of this, I dropped CPI as a feature from the model, which will reduce the collinearity.

Interpretting the Final Linear Regression Results:

After dropping CPI, the other remaining VIF values are below 1.4 and aren't problematic for collinearity in the model.
In this model, the standard error for Production dropped and shows that it is statistically significant along with Crude Price.
Dropping CPI did drop the \(R^2\) value, but this is intuitive given that CPI was highly collinear.
